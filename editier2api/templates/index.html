This is a demo app for a REST API, this API is authencated with token

Need permission for following API call

With GET request to http://127.0.0.1:8000/datalist/, you can get list of data from DB

With POST request to http://127.0.0.1:8000/datalist/, you add more data to the list

Don't need permission for following API call

With and ID, you get perform GET, PUT, DELETE request to the data, http://127.0.0.1:8000/editier2/id
